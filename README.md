## Trả lời câu hỏi
1. So sánh Payload (REST vs GraphQL):
REST API (lấy tất cả) thường trả về dư thừa dữ liệu vì endpoint cố định, nên payload lớn hơn, tốn băng thông và thời gian xử lý.
GraphQL chỉ lấy đúng các trường cần thiết, nên payload nhỏ gọn hơn, giảm dung lượng truyền tải và tối ưu hiệu năng.

2. Thay đổi giá sản phẩm:
Sử dụng Mutation trong GraphQL.
Vì Query chỉ dùng để lấy dữ liệu (read-only), không làm thay đổi dữ liệu.
Còn Mutation được thiết kế để tạo, cập nhật hoặc xóa dữ liệu, nên phù hợp để thay đổi giá sản phẩm.
